# ග්ලැසියර් වල ආර්ථික වටිනාකම (Economic Value of Glaciers)

මෙම යෙදුම මගින් ග්ලැසියර් වල ඇති ආර්ථික වටිනාකම ප්‍රධාන කරුණු තුනක් යටතේ පැහැදිලි කරයි. මෙය ජංගම දුරකථන, වෙබ් සහ පරිගණක යන සියලුම උපාංග වලට සහය දක්වන පරිදි නිර්මාණය කර ඇත.

## ප්‍රධාන කරුණු
1. **මිරිදිය ජල සැපයුම සහ කෘෂිකර්මාන්තය**: වියළි කාලවලදී ගංගා පෝෂණය කරමින් කෘෂිකර්මාන්තයට ජලය සැපයීම.
2. **ජල විදුලි උත්පාදනය**: ග්ලැසියර දියවීමෙන් ලැබෙන ජලයෙන් අඩු පිරිවැයකින් විදුලිය නිපදවීම.
3. **සංචාරක කර්මාන්තය**: ස්වභාවික සෞන්දර්යය සහ හිම ක්‍රීඩා හරහා සංචාරකයින් ආකර්ෂණය කර ආදායම් ඉපයීම.

## තාක්ෂණික තොරතුරු
- Framework: Flutter
- Platforms: iOS, Android, Web, Desktop
